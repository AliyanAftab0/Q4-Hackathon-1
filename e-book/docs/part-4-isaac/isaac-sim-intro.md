# 12. Introduction to Isaac Sim

NVIDIA Isaac Sim is a photorealistic, physically accurate virtual environment for developing AI-based robots.

## Key Features
- **Omniverse**: Built on Pixar's USD, allowing collaborative scene building.
- **GPU Physics**: PhysX 5.0 runs on GPU, simulating thousands of parallel environments.
- **ROS 2 Bridge**: Seamless connectivity to the ROS 2 software stack.

## Installation
Requires an **RTX GPU** (2070 or higher).
1. Install NVIDIA Omniverse Launcher.
2. Download 'Isaac Sim' from the Exchange.
3. Launch via `./isaac-sim.sh`.

## First Steps
- **Load a Robot**: `Create -> Isaac -> Robots -> Ant`.
- **Add Physics**: The stage requires a `PhysicsScene`.
